# Tarot Sources and Method Choices

## Adopted

- Drawing is implemented as a Fisher-Yates shuffle over the selected deck.
- System randomness is the default for real readings.
- Seeded mode is only for tests and reproducible demos.
- The AI interprets cards after the script produces a concrete draw.

Primary method reference:

- Fisher-Yates shuffle: https://en.wikipedia.org/wiki/Fisher%E2%80%93Yates_shuffle

Copyright boundary reference:

- Rider-Waite-Smith copyright discussion: https://sacred-texts.com/tarot/faq.htm

## Not adopted

- No card art is bundled.
- No copyrighted modern deck text is copied into the repository.
- No agent-selected "intuitive" card choice is allowed.
- No claim is made that tarot predicts events with certainty.

## Notes

If future versions add card images, they should use confirmed public-domain sources or user-provided assets with clear licensing.

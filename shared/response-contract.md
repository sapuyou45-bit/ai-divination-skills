# Response Contract

All readings should use this structure unless the user asks for a different format.

## English

1. Result
2. Symbolic Reading
3. Situation Mapping
4. Hidden Variables
5. Actionable Guidance
6. Boundaries

## Chinese

1. Result
2. Symbolic Reading
3. Situation Mapping
4. Hidden Variables
5. Actionable Guidance
6. Boundaries

Use the user's language for the section body. The headings may be translated when that improves readability.

## Rules

- Start from the script output or user-provided cast.
- Do not invent missing cards, lines, positions, or dates.
- Tie every major claim back to the generated result.
- Prefer small, reversible actions over absolute advice.
- Include uncertainty when the evidence is weak.

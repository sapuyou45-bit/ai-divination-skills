Open-source skills for AI agents

Direct divination tools, grounded agent workflows.

Tarot, I Ching, and Xiao Liu Ren skills that separate the divination process from interpretation. Scripts perform the draw or cast and emit JSON. AI only interprets.

View on GitHub Try the scripts

Why it exists

The model should not invent the oracle result.

This project treats divination systems as symbolic reasoning and reflection tools, not deterministic prediction engines. A local script creates the card draw, hexagram, or position; the AI agent interprets that concrete result.

Methodological rigor

Randomness and interpretation stay separate.

For real readings, the divination result comes from system randomness, a traditional probability model, or explicit user-provided physical input. Seeded mode is reserved for tests and reproducible demos.

  • Tarot uses an explicit Fisher-Yates shuffle.
  • I Ching supports three-coin casts and digital yarrow-equivalent probabilities.
  • Xiao Liu Ren prefers lunar numbers, with optional lunar-time conversion.
  • Approximate modes are labeled with warnings.

Quick start

Install once, call one CLI.

Use pip install . from a checkout, then call tarot, I Ching, Xiao Liu Ren, or agent templates through ai-divination.

pip install .
ai-divination tarot --deck major --spread three-card --reversals
ai-divination iching --method yarrow
ai-divination xiaoliuren --method numbers --month 3 --day 12 --hour 7
ai-divination template tarot

Install

Copy only the skills you need.

Each skill is self-contained with a SKILL.md file, OpenAI agent metadata, scripts, and references. 

cp -R skills/tarot ~/.codex/skills/tarot
cp -R skills/iching ~/.codex/skills/iching
cp -R skills/xiaoliuren ~/.codex/skills/xiaoliuren

Safety

Symbolic reflection, not certainty.

  • Do not use readings as medical, legal, financial, or crisis guidance.
  • Tie every interpretation to the generated result.
  • Preserve user agency and offer small, reversible next steps.

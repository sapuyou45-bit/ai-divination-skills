# I Ching Interpretation Template

Use this only after the I Ching script has produced Input JSON.

## Input JSON To Cite

- `method`
- `requested_method`
- `randomness`
- `line_order`
- `method_details`
- `line_probabilities`
- `lines`
- `changing_lines`
- `primary_hexagram`
- `resulting_hexagram`
- `warning`, when present

## Boundaries

- Do not invent line values, changing lines, trigrams, or hexagram names.
- Do not redraw or recast because the primary or resulting hexagram feels inconvenient.
- Do not reverse the line order; script output is bottom-to-top.
- Do not ignore warnings from compatibility aliases or approximate methods.

## Suggested Response Shape

1. Restate the primary hexagram, changing lines, and resulting hexagram.
2. Explain the casting method and probability model when relevant.
3. Read the primary hexagram as the present pattern.
4. Read changing lines as pressure points or transition points.
5. Read the resulting hexagram as the direction of change.
6. Offer bounded guidance tied to the user's question.

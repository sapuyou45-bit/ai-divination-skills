给 AI agent 使用的开源技能

直接、可验证的占卜工具，服务于稳健的 agent 工作流。

塔罗、易经、小六壬技能将占卜过程与解释分开：脚本负责抽取或起课并输出 JSON，AI 只负责解读。

查看 GitHub 试用脚本

为什么做这个

模型不应该自己编出占卜结果。

本项目将占卜体系视为象征推理与反思工具，而不是确定性预言机器。 本地脚本生成牌面、卦象或位置，AI agent 基于这个具体结果进行解释。

方法论严谨性

随机过程和 AI 解读清楚分开。

真实 reading 的占卜结果来自系统随机、传统概率模型，或用户自己提供的实体占卜输入。 seed 只用于测试和可复现 demo。

  • 塔罗使用显式 Fisher-Yates 洗牌。
  • 易经支持三枚硬币法和数字蓍草等价概率。
  • 小六壬优先使用农历数字，也支持可选 lunar-time 转换。
  • 近似模式必须带 warning，不伪装成传统准确起法。

快速开始

安装一次，用一个 CLI 调用。

在 checkout 中使用 pip install .，之后通过 ai-divination 调用塔罗、易经、小六壬或 agent 解读模板。

pip install ai-divination-skills
ai-divination tarot --deck major --spread three-card --reversals
ai-divination iching --method yarrow
ai-divination xiaoliuren --method numbers --month 3 --day 12 --hour 7
ai-divination template tarot

安装

只复制你需要的技能。

每个 skill 都是自包含目录，包含 SKILL.md、OpenAI agent 元数据、脚本和参考资料。 

cp -R skills/tarot ~/.codex/skills/tarot
cp -R skills/iching ~/.codex/skills/iching
cp -R skills/xiaoliuren ~/.codex/skills/xiaoliuren

安全边界

象征性反思，不是确定性预言。

  • 不要把解读用作医疗、法律、金融或危机处置建议。
  • 每个主要解释都要回到脚本生成的具体结果。
  • 保留用户自主性，并给出小而可逆的下一步。
